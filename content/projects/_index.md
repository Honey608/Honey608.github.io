收集互联网上觉得好用、好看、值得反复翻阅的资源。

## 工具

- [Excalidraw](https://excalidraw.com) - 手绘风白板，画架构图、流程图非常顺手，支持实时协作
- [Vite](https://vitejs.dev) - 下一代前端构建工具，极速开发服务器，基于原生 ESM
- [Can I Use](https://caniuse.com) - 查询 HTML/CSS/JS 特性的浏览器兼容性，前端必备
- [RegExr](https://regexr.com) - 在线正则调试工具，实时匹配高亮，附常用正则示例库
- [Squoosh](https://squoosh.app) - 在线图片压缩工具，支持 WebP/AVIF，实时预览压缩效果
- [Ray.so](https://ray.so) - 生成好看的代码截图，多种主题可选
- [transform.tools](https://transform.tools) - 各种格式相互转换，JSON/YAML/SVG/CSS 等

## 文档 & 参考

- [MDN Web Docs](https://developer.mozilla.org/zh-CN/) - Web 技术权威文档，HTML、CSS、JS API 参考
- [Vue.js 官方文档](https://cn.vuejs.org) - Vue 3 中文文档，组合式 API、响应式系统详解
- [React 官方文档](https://zh-hans.react.dev) - React 新版中文文档
- [TypeScript 手册](https://www.typescriptlang.org/zh/docs/) - TypeScript 官方中文文档
- [Nginx 官方文档](https://nginx.org/en/docs/) - Nginx 配置、模块参考

## 学习资源

- [JavaScript.info](https://zh.javascript.info) - 现代 JS 教程，从基础到高阶，有中文翻译
- [CSS Tricks](https://css-tricks.com) - 大量 CSS 布局、动画实战教程和深度文章
- [freeCodeCamp](https://www.freecodecamp.org/chinese) - 免费编程学习路径，包含项目实践
- [Roadmap.sh](https://roadmap.sh) - 各种开发方向的学习路线图
- [算法可视化](https://visualgo.net/zh) - 数据结构与算法动态可视化

## 新闻 & 阅读

- [Hacker News](https://news.ycombinator.com) - 技术社区最经典的诚愿度议论区
- [Paul Graham](https://paulgraham.com/articles.html) - YC 创始人，创业、写作、编程层面深度文章
- [Wait But Why](https://waitbutwhy.com) - Tim Urban 的博客，大量手绘插图，选题深刻有趣
- [ruanyifeng.com](https://www.ruanyifeng.com/blog/) - 阔读周刊作者阿里的博客，技术漫谈均有

## 设计 & 视觉

- [Dribbble](https://dribbble.com) - 设计师作品展示社区
- [Awwwards](https://www.awwwards.com) - 优秀网页设计案例展示
- [Coolors](https://coolors.co) - 配色方案生成器，快速找到符合的配色
- [Google Fonts](https://fonts.google.com) - 免费字体库，大量中英文字体
