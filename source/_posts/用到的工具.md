</font><font style="color:rgb(0, 0, 0);">安装  
</font><font style="color:rgb(0, 0, 0);">一、手动解压安装包：  
</font><font style="color:rgb(0, 0, 0);">1、在user目录下新建java文件夹：</font>

```shell
# cd /usr/
# mkdir java
# cd java
```

<font style="color:rgb(0, 0, 0);">2.下载jdk1.8</font>

```shell
# wget http://download.oracle.com/otn-pub/java/jdk/8u181-b13/96a7b8442fe848ef90c96a2fad6ed6d1/jdk-8u181-linux-x64.tar.gz?AuthParam=1534129356_6b3ac55c6a38ba5a54c912855deb6a22
```

<font style="color:rgb(0, 0, 0);">3、解压：使用tar -zxvf 文件名进行解压</font>

<font style="color:rgb(0, 0, 0);">解压之后：文件夹名称jdk1.8....</font>

<font style="color:rgb(0, 0, 0);">4、配置环境变量：</font>

```shell
# vi /etc/profile
```

<font style="color:rgb(0, 0, 0);">将如下配置添加至文件中，然后保存退出。</font>

```bash
#java
export JAVA_HOME=/usr/java/jdk8u282-b08
export PATH=$JAVA_HOME/bin:$PATH
export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib
```

<font style="color:rgb(0, 0, 0);">5、让刚刚设置的环境变量生效并检查是否安装成功  
</font><font style="color:rgb(0, 0, 0);">①让刚刚设置的环境变量生效</font>

```bash
source /etc/profile或 . /etc/profile
```

</font><font style="color:rgb(0, 0, 0);">java -version</font>
